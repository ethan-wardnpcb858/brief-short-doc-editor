## What Brief is for

Brief is built for quick document work in the browser. The experience stays intentionally lean, helping you go from a rough idea to finished text without extra setup or distracting complexity.

It fits short-form drafting, rapid editing passes, and writing tasks where efficiency matters. If you need a stripped-down web editor that keeps the focus on text instead of tooling, Brief gives you a straightforward place to work.
